# FAQ

## What is DEGEN EMPEROR TERMINAL?
DEGEN EMPEROR TERMINAL is a closed-core crypto trading terminal for market scanning, trade planning, and execution workflows.

## Is this an open-source trading bot?  
No. This repository is public documentation only. The core engine is private.

## Does it support auto trading?
Yes. The terminal includes auto-execution capabilities as part of the private system.

## Does it work with CEX and DEX platforms?
Yes. The system supports exchange connectivity across centralized and decentralized trading environments.

## What market data does the screener use?
The public documentation covers volume, open interest, funding rates, and multi-timeframe RSI logic.

## Is this financial advice?
No. This project is a software product and documentation layer, not financial advice.
