# Access Model

DEGEN EMPEROR TERMINAL is a **closed-core** trading product.

## Public layer
This GitHub repository contains:

- product overview
- architecture notes
- market screener documentation
- screenshots
- FAQs
- public roadmap

## Private layer
The following components are not public:

- signal generation engine
- execution logic
- exchange connection internals
- proprietary strategy logic
- private deployment setup

## Why the core is private
The terminal is designed as a practical execution system, not as an open-source indicator pack.

Publishing the core would destroy the product edge and compromise security.

## Access direction
Access to the full system may be limited, curated, or subscription-based in future versions.
